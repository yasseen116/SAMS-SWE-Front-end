Undergraduate Courses

Curriculum designed to build strong foundations in software engineering and modern development practices.

Introduction to Software Engineering
SE101
3 credit hours
Overview of software engineering principles, development life cycles, process models, requirements basics, and team collaboration.
Prerequisites: None
Object-Oriented Programming
SE120
3 credit hours
Principles of abstraction, encapsulation, inheritance, polymorphism, and design of robust OOP applications using Java or C#.
Prerequisites: Programming Fundamentals
Data Structures & Algorithms
SE210
3 credit hours
Core data structures, algorithm analysis, problem solving, and complexity concepts crucial for efficient software systems.
Prerequisites: Object-Oriented Programming
Software Requirements Engineering
SE220
3 credit hours
Requirements elicitation, analysis, specification, validation, stakeholder management, and documentation techniques.
Prerequisites: Introduction to Software Engineering
Software Design & Architecture
SE310
3 credit hours
Design principles, patterns, architectural styles, quality attributes, and modeling with UML for large-scale systems.
Prerequisites: Software Requirements Engineering
Software Testing & Quality Assurance
SE330
3 credit hours
Test design, automation, coverage analysis, continuous integration, and quality metrics to ensure reliable software delivery.
Prerequisites: Software Design & Architecture
Web Application Development
SE340
3 credit hours
Frontend frameworks, backend services, RESTful APIs, authentication, and deployment for modern web applications.
Prerequisites: Object-Oriented Programming
DevOps & Cloud Engineering
SE360
3 credit hours
CI/CD pipelines, containerization, infrastructure-as-code, monitoring, and cloud platforms (AWS/Azure/GCP).
Prerequisites: Software Testing & Quality Assurance
Databases & Information Systems
SE250
3 credit hours
Relational modeling, SQL, transactions, normalization, and intro to NoSQL for scalable data-driven applications.
Prerequisites: Object-Oriented Programming
Software Project Management
SE420
3 credit hours
Planning, estimation, risk management, agile methodologies, team leadership, and delivery of software projects.
Prerequisites: Software Design & Architecture
