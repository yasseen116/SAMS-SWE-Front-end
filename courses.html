Software Engineering Courses

Undergraduate Courses

Curriculum designed to build strong foundations in software engineering and modern development practices.

Year 1 Courses

Semester 1

Introduction to Software Engineering
SE101
3 credit hours
Overview of software engineering principles, development life cycles, process models, requirements basics, and team collaboration.
Prerequisites: None
Object-Oriented Programming
SE120
3 credit hours
Principles of abstraction, encapsulation, inheritance, polymorphism, and design of robust OOP applications using Java or C#.
Prerequisites: Programming Fundamentals

Semester 2

Data Structures & Algorithms
SE210
3 credit hours
Core data structures, algorithm analysis, problem solving, and complexity concepts crucial for efficient software systems.
Prerequisites: Object-Oriented Programming
Software Requirements Engineering
SE220
3 credit hours
Requirements elicitation, analysis, specification, validation, stakeholder management, and documentation techniques.
Prerequisites: Introduction to Software Engineering
Databases & Information Systems
SE250
3 credit hours
Relational modeling, SQL, transactions, normalization, and intro to NoSQL for scalable data-driven applications.
Prerequisites: Object-Oriented Programming

Year 2 Courses

Semester 1

Software Design & Patterns
SE310
3 credit hours
SOLID principles, design patterns, UML modeling, architectural styles, and best practices for scalable systems.
Prerequisites: Software Requirements Engineering
Web Development Technologies
SE340
3 credit hours
HTML5, CSS3, JavaScript, responsive design, web APIs, and front-end frameworks for modern web applications.
Prerequisites: Object-Oriented Programming

Semester 2

Software Testing & QA
SE330
3 credit hours
Testing strategies, unit testing, integration testing, automation frameworks, and quality assurance best practices.
Prerequisites: Software Design & Patterns
Software Project Management
SE360
3 credit hours
Project planning, scheduling, resource allocation, risk management, agile methodologies, and team coordination.
Prerequisites: Software Testing & QA
Software Capstone Project
SE420
4 credit hours
Comprehensive capstone project integrating all learned concepts, real-world application development, and professional practices.
Prerequisites: Software Design & Patterns
